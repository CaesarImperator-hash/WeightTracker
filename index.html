Liens entre tes données

Le chiffre (r) va de -1 à +1 : proche de 0 = pas de lien, proche de +1 ou -1 = lien fort. Calculé sur tous les jours où les deux données existent.

Une corrélation indique que deux choses évoluent ensemble — pas que l'une cause l'autre. 🙂 Et comme le poids reflète plusieurs jours de fond (eau, digestion...), un lien jour pour jour avec le poids est souvent plus faible que l'effet réel, qui se voit surtout sur plusieurs semaines.
